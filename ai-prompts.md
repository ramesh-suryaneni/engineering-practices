**The 4 Pillar Framework:**

1. *Persona (Who)* - Tell AI who it should be
2. *Task (What)* - State exactly what you want
3. *Context (Why)* - Provide all background information
4. *Format (How)* - Specify how you want the response


**Ready-to-Use Templates:**

*For Content Creation:*

You are a [ROLE].
Write a [CONTENT TYPE] about [TOPIC].
Audience: [Describe Audience]
Tone: [Formal/Casual/Witty]
Length: [500 words, 3 paragraphs]
Format: [Blog Post, Social Media Post]

*For Code Generation:*

You are an expert [PROGRAMMING LANGUAGE] developer.
Task: [Specific coding task]
Requirements: [List requirements]
Constraints: [List constraints]
Provide clean, commented code"

*For Analysis Tasks:*

Analyze the following [TEXT/DATA]:
[Insert content]
Focus on: [List aspects]
Provide your analysis in a [FORMAT]."
